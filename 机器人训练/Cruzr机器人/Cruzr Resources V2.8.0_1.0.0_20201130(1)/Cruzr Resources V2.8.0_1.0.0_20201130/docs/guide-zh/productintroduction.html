    Cruzr产品介绍

    ​ Cruzr云平台商用服务机器人，是优必选科技自主研发的一款类人形机器人。具有灵活的自由度，语音及视觉能力，立体导航和避障能力。

    ​ Cruzr可服务于交通出行、政务大厅、购物中心、银行、星级酒店、展览馆、4S店、医院、商业地产等一系列场景, 可以有效地简化工作流程、降低人力成本、挖掘商业价值、减少运营成本，并帮助企业和服务大厅智能转型。

    产品参数

    项目 参数规格
    产品尺寸 1210(H)630(W)537(D)mm
    产品净重 约45kg
    屏幕 11.5寸TFT高清屏
    分辨率 1920*1080像素
    摄像头 1300万像素高清摄像头
    电池 电池类型：铁锂电池，容量：25Ah，电压：25.6V
    最大越障高度 1cm
    最大爬坡角度 6度
    行进速度 0.2/0.5/0.7 m/s (最大速度1m/s)
    通信 Wi-Fi（2.4G/5G），支持外接4G
    喇叭 85dB（正前方60cm处）
    工作温度 5-25摄氏度
    工作环境 室内
    系统平台 Aandroid（应用层）+ROS（控制层）
    传感器 头部：6+0麦克风阵列 *1
    腰部：深度摄像头*1
    手臂：电子皮肤2*2（双臂，2块/手臂）
    底盘：超声传感器 6，红外传感器 * 9，9轴陀螺仪传感器 1，激光雷达 1，温湿度传感器 1
