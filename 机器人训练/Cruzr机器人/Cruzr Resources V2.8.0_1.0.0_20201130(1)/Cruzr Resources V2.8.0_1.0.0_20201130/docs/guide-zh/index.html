    2020.2.12 发布Cruzr开放平台V2.5.3版本SDK和说明文档
    2020.2.21 修改说明文档
    开发你的第一个机器人应用
    添加共享库声明，修改为android:name="com.ubtrobot.rosa"
    引入sdk包，修改目录位置为cruzr-libs
    2020.3.24 云端API上线（仅国内）
    版本号：V1.0.0.20200324_release
    本次发布的版本API包括：
    1.鉴权与授权
    2.查询机器人基础信息
    3.查询机器人诊断信息
    4.查询人脸库
    5.查询活跃信息
    6.查询语音信息
    2020.4.10 发布Cruzr开放平台V2.7.0版本SDK和说明文档
    1.增加空闲服务，多媒体服务，用户管理，视觉服务相关接口和说明。
    2.设置：新增设置机器人语言
    3.语音助手：新增语音助手各组件单独显示和隐藏，引导问法显示和隐藏的接口
    4.补充传感器清单，诊断信息清单等
    2020.11.10 发布Cruzr开放平台V2.8.0版本SDK和说明文档
    1.语音助手：新增语音助手全局显示隐藏接口。
    2.视觉服务：新增人脸检测、人脸追踪、离线图片识别 、在线图片识别 、人脸抓拍、开启关闭视觉服务接口。
    3.用户管理：新增用户注册接口。
    4.FAQ：添加使用google语音服务常见流程。
