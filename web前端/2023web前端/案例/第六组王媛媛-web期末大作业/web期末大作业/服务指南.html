服务指南

读者须知与办证指南

读者阅览图书、搜寻资讯及参与图书馆活动的公共文化场所，公共资源的有限性与大众需求。

服务区布局

深圳图书馆的布局由3D动态图组成，可以更直接更详细了解图书馆的整体规划。

书刊借阅服务

实行“全面开放、免证进馆、分层管理、一卡通行”方式，全面推行开架阅览、自助服务。

网络与数字资源服务

深圳图书馆现拥有数据库46个，包括电子期刊2398974册，电子图书1327061种；本馆自主建设并对外发布的地方文献数据库容量52.14G，65000条数据。均免费提供服务。

参考资询与专题服务

参考信息服务提供的服务有常见问题、问图书馆员、课题服务等。可通过多种在线方式咨询。

少儿服务

为13周岁以下少年儿童提供办证、图书借还、阅读指导服务，以及组织阅读推广活动等。

阅读推广服务

致力于全民阅读的推广，优化并完善服务平台，促进阅读理念传播，倡导爱读书，营造阅读文化氛围。

语音/电话系统

可以获取图书馆每个服务区域的联系电话。

移动服务门户

移动图书馆以深圳文献港为基础，对各个图书馆的中外文图书、期刊、报纸、学位论文、标准、专利等文献进行了全面整合。

短信服务平台

持“深圳图书馆读者证”读者均可通过定制，享受手机短信服务。
